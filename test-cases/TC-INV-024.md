# TC-INV-024

## Feature
Delete Item

## Title
Verify deletion / inactivation of an inventory item

## Objective
Verify that clicking the delete icon deactivates an item and updates its status.

## Preconditions
User is on `/newui/Inventory/Items`.

## Test Steps
1. Click the "Delete Item" icon next to an item in the list.
2. Confirm the action in the popover / confirmation dialog.

## Expected Result
The item is deactivated/deleted, and its status is updated (or it is removed from the active items list).

High

## Automation Candidate
