# TC-INV-025

## Feature
Duplicate Item

## Title
Verify duplicating an existing item

## Objective
Confirm that clicking the duplicate icon pre-fills the Add Item form with details from the selected item.

## Preconditions
User is on `/newui/Inventory/Items`.

## Test Steps
1. Click the "Duplicate Item" icon next to an item in the list.

## Expected Result
The Add Item form is opened and pre-filled with all details of the selected item (except SKU which is left blank or appended with a suffix for uniqueness).

## Priority
Medium

## Automation Candidate
Yes
