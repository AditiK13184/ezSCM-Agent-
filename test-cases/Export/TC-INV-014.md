# TC-INV-014

## Feature
Export

## Title
Verify Download Inventory Report functionality

## Objective
Ensure that clicking the download icon next to the search input exports the inventory report.

## Preconditions
User is on `/newui/user/inventory/reports`.

## Test Steps
1. Click the button with the title "Download Inventory Report".
    2. Wait for the download file to be created.

## Expected Result
*   An export action is triggered and a file (typically CSV or Excel) containing the inventory report starts downloading.

## Priority
High

## Automation Candidate
Yes
